Synonimical:
Array.range( 10                ) = []
Array.range(  0,     9         ) = []
Array.range(  0,     9,     1  ) = []

Single items:
Array.range( 30,    30         ) = []
Array.range(-30,   -30         ) = []

Different ranges:
Array.range(  5,    10         ) = []
Array.range(  1,    10,     3  ) = []
Array.range(-10,   -90,   -30  ) = []
Array.range(  1.2,  10.3,   3.1) = []
Array.range( -1.5,   1.5,      ) = []

Illegal usage:
Array.range(-10,   -90,    30  ) = []
Array.range(                   ) = []
